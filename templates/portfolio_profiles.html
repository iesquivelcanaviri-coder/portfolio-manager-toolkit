Portfolio Profiles and Portfolio Manager Action Centre

This page is designed as a portfolio manager review screen. It does not only describe each client profile — it shows the type of checks, judgment, and implementation actions a manager would complete before approving a trade, sizing a position, or rejecting an idea.

The purpose is to move from profile review to investment action: assess mandate fit, identify risk and compliance issues, and decide what must happen before any position is added to the portfolio.

Before approving any trade, the manager should decide whether the proposed security improves the portfolio in a way that is consistent with the stated objective, benchmark, risk profile, liquidity needs, behavioural considerations, and formal mandate restrictions.

  • Does this idea fit the client objective and benchmark?
  • Does the security sit inside the allowed mandate and product universe?
  • Would the position breach any weight, issuer, or concentration rule?
  • Does the client’s liquidity need make this allocation impractical?
  • Is the currency exposure acceptable for this mandate?
  • Would this position create behavioural stress during drawdowns?
  • Does the reporting frequency support the complexity of the proposed idea?
  • Confirm suitability before trade approval
  • Check max position size before order entry
  • Review mandate restrictions before execution
  • Review liquidity and cash needs before funding the trade
  • Confirm benchmark consistency and portfolio role
  • Document rationale for approval, limitation, or rejection
  • Set monitoring frequency based on mandate and risk level
