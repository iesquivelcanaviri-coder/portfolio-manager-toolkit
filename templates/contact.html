This project was developed as a portfolio management decision-support application using Flask, HTML, CSS, JavaScript, and Yahoo Finance data.

It demonstrates how a portfolio manager can combine client mandate information, investment decision criteria, market analysis, and portfolio construction logic into a simple workflow.

Project Focus

  • Portfolio-specific suitability analysis
  • Risk and return assessment
  • Position sizing decisions
  • Market screening and ranking
  • Portfolio construction support
