Portfolio Criteria

  • Expected return should be attractive relative to risk.
  • Volatility should be appropriate for the portfolio objective.
  • Beta should be consistent with the benchmark and mandate.
  • The stock or ETF should justify its portfolio weight.
  • Risk-adjusted return should support the investment case.
  • The final decision should map to a realistic portfolio size.

The Math Behind a 6% Position

1. Expected Return (ER)

Formula: ER = (Target Price − Current Price) / Current Price

How it is applied: A high weight requires high expected return.

2. Risk-Adjusted Return (RAR)

Formula: RAR = ER / σ

How it is applied: Return must justify volatility.

3. Correlation to Portfolio

Formula: Marginal Risk = w · σ · ρ

How it is applied: Avoid concentration risk.

4. Contribution to Portfolio Risk

Formula: Risk Contribution = w · β

How it is applied: Keeps portfolio beta stable.

Qualitative Criteria

  • High-quality company
  • Stable earnings
  • No major catalyst
  • Lower risk
  • Fits long-term strategy
