Portfolio Criteria

  • Expected return should be attractive relative to risk.
  • Volatility should be appropriate for the portfolio objective.
  • Beta should be consistent with the benchmark and mandate.
  • The stock or ETF should justify its portfolio weight.
  • Risk-adjusted return should support the investment case.
  • The final decision should map to a realistic portfolio size.

Qualitative Criteria

  • High-quality business
  • Predictable earnings
  • Clear catalyst
  • Low-to-moderate volatility
  • High liquidity
  • Portfolio manager understands the company deeply

The Math Behind a 6% Position

1. Expected Return (ER)

Formula: ER = (Target Price − Current Price) / Current Price

How it is applied: A 6% position usually requires a high expected return, often around 20–40%, because the portfolio manager is allocating meaningful capital to a high-conviction idea.

2. Risk-Adjusted Return (RAR)

Formula: RAR = ER / σ

How it is applied: σ represents stock volatility. A stock can only justify a 6% weight if return is attractive relative to risk, not just attractive in absolute terms.

3. Correlation to Portfolio

Formula: Marginal Risk = w · σ · ρ

How it is applied: w is position weight and ρ is correlation to the portfolio. Even a strong stock should not receive a 6% weight if it pushes portfolio risk too high.

4. Contribution to Portfolio Risk

Formula: Risk Contribution = w · β

How it is applied: A 6% position must still keep the overall portfolio beta near target, for example around 1.0, so the holding does not distort the mandate.

Qualitative Criteria

  • High-quality company
  • Stable earnings
  • No major catalyst
  • Lower risk
  • Fits long-term strategy

The Math Behind a 3.5% Position

1. Moderate Expected Return

Formula: ER = (Target Price − Current Price) / Current Price

How it is applied: A core holding usually has a moderate expected return, often around 10–20%, rather than the very high return required for a top conviction position.

2. Lower Volatility

Formula: σ < Portfolio Average

How it is applied: A core holding should usually be less volatile than the more aggressive names in the portfolio, supporting stability and consistency.

3. Correlation / Portfolio Risk Alignment

Formula: Correlation = ρ

How it is applied: If the stock is highly correlated with the rest of the portfolio, the manager may keep the weight smaller to avoid concentration of the same risk factor.

4. Balanced Risk Contribution

Formula: Risk Contribution ≈ 3%–5% of total portfolio risk

How it is applied: A 3.5% holding should contribute to portfolio returns without becoming one of the dominant sources of risk.

Qualitative Criteria

  • Adds sector or geographic diversification
  • Higher uncertainty
  • Smaller company
  • Thematic exposure

The Math Behind a 3% Position

1. Higher Volatility

Formula: σ > Portfolio Average

How it is applied: Because volatility is above average, the weight is reduced so the position can still be included without dominating risk.

2. Lower Expected Return Certainty

Formula: ER = (Target Price − Current Price) / Current Price

How it is applied: Expected return may still be attractive, but confidence in the estimate is lower, so the portfolio manager allocates less capital.

3. Diversification Benefit

Formula: ρ < 0.5

How it is applied: A portfolio manager may still accept a 3% position in a volatile stock if its correlation with the rest of the portfolio is low and it improves diversification.

4. Controlled Risk Contribution

Formula: Risk Contribution ≈ 2%–3%

How it is applied: The position should help diversify the portfolio while keeping its share of total portfolio risk contained.

Qualitative Criteria

  • Very high volatility
  • Early-stage company
  • Unproven thesis
  • Illiquid
  • Portfolio manager wants exposure but limited risk

The Math Behind a 1–2% Position

1. Very High Volatility

Formula: σ ≫ Portfolio Average

How it is applied: The position size is kept very small because the stock’s risk is much higher than the overall portfolio average.

2. High Beta

Formula: β > 1.3

How it is applied: A high-beta stock is more sensitive to market moves, so a smaller weight protects the portfolio from excessive swings.

3. High Uncertainty in Expected Return

Formula: ER = (Target Price − Current Price) / Current Price

How it is applied: Even if upside looks attractive, uncertainty around that estimate means the manager reduces position size to control downside risk.

4. Minimal Risk Contribution

Formula: Risk Contribution < 1%

How it is applied: Exploratory positions are designed to give exposure while ensuring that, if wrong, they do not materially damage the full portfolio.

Qualitative Criteria

  • Flexibility
  • Lower volatility
  • Dry powder for opportunities

The Math Behind Cash Allocation

1. Reduce Portfolio Volatility

Formula: σportfolio = Σwi2σi2 + 2Σwiwjσiσjρij

How it is applied: Cash has σ = 0 and ρ = 0, so allocating part of the portfolio to cash reduces total portfolio volatility mathematically.

2. Reduce Drawdowns

Formula: Cash lowers portfolio sensitivity to market declines

How it is applied: Because part of the portfolio is not exposed to risky assets, maximum drawdowns are smaller during stressed markets.

3. Optionality Value

Formula: Cash provides immediate deployable capital

How it is applied: Holding cash allows the portfolio manager to buy attractive assets when volatility spikes or when markets dislocate.
