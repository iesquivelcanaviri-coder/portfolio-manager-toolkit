Portfolio Management Decision Support Tool

This project simulates how a portfolio manager can evaluate securities using return analysis, volatility, Sharpe-like metrics, ranking logic, and internal portfolio sizing criteria. The tool combines Yahoo Finance market data with portfolio management rules to support investment decisions.

What This Project Does

The site is designed to help the user assess securities and portfolio ideas using a structured portfolio management approach. It combines qualitative judgment with quantitative analysis to support decisions on suitability, conviction level, and recommended portfolio weight.

  • Understand the position sizing framework used in the project
  • Review high-level investment decision criteria
  • Analyse a selected stock or ETF using 1-year and 3-month data
  • Compare multiple securities in the market dashboard
  • Filter and sort securities by ticker, decision, score, volatility, Sharpe-like ratio, and ranking score
  • See latest price, expected return, volatility, Sharpe-like ratio, score, and ranking score
  • Apply position sizing rules (6%, 3.5%, 3%, 1–2%, and cash)
  • Build and review portfolio holdings
  • Track portfolio allocation, remaining cash, and average beta

Who This Is For

  • Students learning portfolio management
  • Junior analysts and portfolio managers
  • Advisors needing a simple decision-support tool

How to Use the Site

  1. Start on the Market Dashboard page to screen and compare securities.
  2. Go to the Criteria page to analyse a specific stock or ETF within the context of a chosen portfolio mandate.
  3. Review score, decision, expected return, volatility, Sharpe-like ratio, latest price, and forecast outputs.
  4. Use the position sizing framework to decide whether the security belongs as a high-conviction, core, satellite, exploratory, or cash decision.
  5. Add or update the position and monitor portfolio allocation summary.
